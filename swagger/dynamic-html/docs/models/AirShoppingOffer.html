AirShoppingOffer

  • benefits : Benefit
  • createdAt : Long
    Offer created at unix time
  • disclosures : List
    List of disclosures corresponding to offer.
  • error : String
    Error from provider
  • flights : List
    Flight Segment information. Including departure, arrival and airline information.
  • maxNumberOfStops : Long
    Max number of stops
  • offerID : String
    Offer identifier
  • offerType : String
    Offer type
    Enum:
    oneWay
    roundTrip
    multiCity
  • owner : String
    2 letter abbreviation of airline
  • paymentTimeLimit : String
    Offer payment time limit, after this time created order will become expired
  • price : Price
  • priceBreakdown : PriceBreakdown
  • provider : String
    2 letter abbreviation of airline
