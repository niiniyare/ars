OrderChangePayload

  • action : String
    A flag used in some cases of the OrderChange like accept the disruption of an order or split the order
  • id : String
    AirGateway order ID
  • loyaltyProgramAccount : String
    Loyalty program account number for corporates
  • passengerUpdate : Passenger
  • passengers : List
    List of passengers
  • payment : Payment
  • reshopOfferID : String
    Reshop offer ID from OrderReshop response
  • services : List
    List of service objects
