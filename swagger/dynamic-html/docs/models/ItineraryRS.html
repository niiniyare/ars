ItineraryRS

  • createdAt : Long
    Created at unix time
  • flights : List
    route details
  • maxNumberOfStops : Long
    Max number of stops
  • offerType : String
    Offer type
    Enum:
    oneWay
    roundTrip
    multiCity
  • options : List
    possible fare options for itinerary
  • owner : String
    2-letter code of provider
  • paymentTimeLimit : String
    Payment time limit, after this time created order will become expired
